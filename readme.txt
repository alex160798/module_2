Проект 3 (Юнит 2: Разведывательный анализ данных)

Задачи проекта: 
 - Просмотреть данные.
 - Проверить их на пустые значения.
 - Проверить их на дублированные/полностью скоррелированные значения.
 - Проверить их на наличие выбросов.
 - Отобрать пригодные для дальнейшего построения модели.
 - Повторить если необходимо.

Краткая информация о данных:
Данные представленны в виде файла формата .csv, который содержит информацию об учащихся 15-22
лет, а также их баллы за госэкзамен по математике, которые, как предполагается, будет предсказы-
вать в дальнейшем наша модель.

Этапы работы:
 - Прочитан файл, данные занесены в датафрейм math;
 - Исследовано содержание колонок, отмечено наличие в них Null-значениий;
 - Написаны функции для анализа номинативных и численных показателей;
 - Проведен анализ каждой отдельной колонки, отброшены выбросы (где необходимо), сделаны краткие
выводы для каждого;
 - Проведен анализ численных показателей на коэффициен корреляции;
 - Проведен анализ номинативных показателей на значимость с помощью теста Стьюдента;
 - Отобраны колонки для дальнейшего построения модели;
 - Сделаны выводы.
